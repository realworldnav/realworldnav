express.ui.update_accordion

express.ui.update_accordion(id, *, show, session=None)

Dynamically set accordions' states.

Dynamically (i.e., programmatically) update/modify accordions in a Shiny app. These functions require an id to be provided to the accordion and must also be called within an active Shiny session.

Parameters

id : str

A string that matches an existing accordionâ€™s id.

show : bool | str | list[str]

Either a string or list of strings (used to identify particular accordion_panel(s) by their value) or a bool to set the state of all panels.

session : Optional[Session] = None

A Shiny session object (the default should almost always be used).

            f"Some narrative for section {letter}"
